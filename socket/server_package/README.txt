# IoT 센서 서버 패키지

## 실행 방법

1. IoT_Sensor_Server.exe 실행
2. 서버가 자동으로 RSA 키 생성 (private.pem, public.pem)
3. 포트 12351에서 클라이언트 연결 대기

## 기능

- 멀티 클라이언트 동시 연결 지원
- RSA-2048 암호화 통신
- 네트워크를 통한 공개키 자동 배포
- MySQL 데이터베이스 연동
- 센서 데이터 검증 및 저장

## 주의사항

- MySQL 데이터베이스가 설정되어 있어야 합니다
- 방화벽에서 포트 12351을 허용해주세요
- 첫 실행 시 RSA 키가 자동 생성됩니다

## 독립 실행

- 이 실행 파일은 완전 독립적입니다
- 다른 폴더나 파일 없이도 실행 가능
- 다른 기기에 복사해서 바로 실행 가능

## 생성 파일

실행 후 다음 파일들이 생성됩니다:
- private.pem: 서버 개인키 (보안 중요)
- public.pem: 공개키 (클라이언트가 자동으로 요청)
