# IoT 센서 클라이언트 패키지

## 실행 방법

1. config.json에서 서버 IP 주소 설정
2. IoT_Sensor_Client.exe 실행
3. 서버에서 자동으로 공개키 받아옴
4. 암호화된 센서 데이터를 주기적으로 전송

## 설정 파일 (config.json)

```json
{
    "server": {
        "address": "192.168.1.100",  // 서버 IP 주소
        "port": 12351               // 서버 포트
    },
    "client": {
        "sensor_id": "SENSOR_001",  // 고유 센서 ID
        "send_interval": 10,        // 전송 간격(초)
        "public_key_path": "public.pem"
    }
}
```

## 센서 ID 설정

각 클라이언트마다 고유한 sensor_id를 설정하세요:
- SENSOR_A01, SENSOR_B01, SENSOR_C01 등
- 서버 데이터베이스에 미리 등록되어 있어야 함

## 네트워크 요구사항

- 서버와 네트워크 연결 필요
- 포트 12351 접근 가능해야 함
- 공개키를 자동으로 서버에서 다운로드

## 독립 실행

- 이 실행 파일은 완전 독립적입니다
- config.json과 함께 다른 기기에 복사해서 바로 실행 가능
- 별도 설치나 의존성 없음

## 다중 클라이언트

여러 기기에서 동시 실행 가능:
1. 각 기기에 이 패키지 복사
2. 각각 다른 sensor_id 설정
3. 동일한 서버 주소 설정
4. 독립적으로 실행
